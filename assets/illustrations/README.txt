Coloca aqui las ilustraciones para las fichas.

Archivos sugeridos para modo automatico:
- sumas.svg
- restas.svg
- multiplicaciones.svg
- divisiones.svg
- valor-posicional.svg

Recomendaciones:
- Formato recomendado: SVG (ideal) o PNG transparente.
- Tamano sugerido: entre 600x600 y 1200x1200.
- Evita fondos muy oscuros: se renderiza como marca de agua.

Modo manual:
- En la UI puedes escribir un archivo (por ejemplo "mi-dibujo.svg").
- Si escribes solo el nombre, se busca en assets/illustrations/.
- Si pones una ruta (ej: assets/illustrations/x.svg) tambien funciona.
